# Amazon Clone 
A responsive Amazon-style e-commerce interface clone built using HTML, CSS, and JavaScript, allowing users to browse products and manage a shopping cart with persistent storage using LocalStorage. This project demonstrates front-end UI replication, DOM manipulation, and interactive cart functionality.

## Features
- Amazon-style product browsing interface
- Add, remove and update items in a cart
- Persistent car storage using local storage
- Dynamic cart tool calculations
- Responsive layout for desktops and mobile views
- Interactive UIO updates through DOM manipulation

## Tech Stack 
- JavaScript
- HTML
- CSS
- Local Storage
